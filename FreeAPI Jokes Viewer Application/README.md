# FreeAPI Jokes Viewer Application

A modern, interactive web application that fetches and displays random jokes from the FreeAPI endpoint. Built with React, Vite, and pure CSS for a clean, professional user experience.

Features

## ✨ Interactive Joke Browsing

Fetch 10 jokes at a time from FreeAPI
Navigate between jokes with Previous/Next buttons
Auto-load new jokes when reaching the end

## ❤️ Favorites System

Save your favorite jokes with one click
View all favorites in a dedicated section
Remove favorites anytime

## 🎨 Professional Design

Clean, modern UI with gradient accents
Responsive design for desktop, tablet, and mobile
Smooth animations and transitions
Hover effects and interactive states

## ⚡ Performance

Fast load times with Vite bundler
Optimized React rendering
Lightweight CSS (no framework dependencies)

## 🛠 Tech Stack

Frontend: React 19.2.5 with React DOM
Build Tool: Vite 8.0.10
Styling: Pure CSS with CSS Variables
State Management: React Hooks (useState, useEffect)
API: FreeAPI Random Jokes Endpoint
